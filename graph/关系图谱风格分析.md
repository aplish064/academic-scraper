# LLM Wiki 关系图谱风格分析与定制指南

## 📊 概述

LLM Wiki 使用 D3.js 力导向图（Force-Directed Graph）来展示知识图谱中的关系网络。其设计风格融合了现代 UI 设计趋势，包括毛玻璃效果、粒子背景、渐变色彩和流畅的交互动画。

```typescript
// 节点大小根据连接数动态调整
```

### 3. **物理模拟**

#### 力场配置
```typescript
const sim = d3force.forceSimulation<GraphNode>(nodes)
  .force("link", d3force.forceLink(links)
    .distance(170)      // 链接长度
    .strength(0.22))    // 链接强度
  .force("charge", d3force.forceManyBody()
    .strength(-650)     // 斥力强度
    .distanceMax(900))  // 最大作用距离
  .force("center", d3force.forceCenter(width / 2, height / 2))
  .force("collision", d3force.forceCollide()
    .radius((d) => radius(d) + 14)
    .strength(0.9))
  .force("x", d3force.forceX(width / 2).strength(0.02))
  .force("y", d3force.forceY(height / 2).strength(0.02))
  .alphaDecay(0.005)    // 冷却速度（慢）
  .velocityDecay(0.28)  // 速度衰减
  .alphaTarget(0.015);  // 永不停止，保持微动
```

#### 环境噪声（呼吸效果）
```typescript
// 每个时间步添加微小随机速度
sim.force("noise", () => {
  for (const n of nodes) {
    n.vx = (n.vx ?? 0) + (Math.random() - 0.5) * 0.09;
    n.vy = (n.vy ?? 0) + (Math.random() - 0.5) * 0.09;
  }
});
```

### 4. **粒子背景**

#### 设计理念
- **加法混合**：重叠粒子产生发光效果
- **径向渐变**：每个粒子有柔和光晕
- **正弦闪烁**：透明度按正弦波变化

#### 调色板
```typescript
const PALETTE = [

  [203, 166, 247], // mauve
  [148, 226, 213], // teal

];
```

#### 渲染代码
```typescript
const grad = ctx.createRadialGradient(p.x, p.y, 0, p.x, p.y, p.r * 7);
grad.addColorStop(0, `rgba(${r},${g},${b},${alpha})`);
grad.addColorStop(0.4, `rgba(${r},${g},${b},${alpha * 0.3})`);
```

### 5. **交互设计**

#### 悬停高亮
```typescript
nodeSel.on("mouseenter", function (_event, d) {
  const neighbors = adjacency.get(d.id) ?? new Set();

  // 非相关节点变暗
  nodeSel.classed("dim", (n) => n.id !== d.id && !neighbors.has(n.id));

  // 当前节点和邻居高亮
  nodeSel.classed("highlight", (n) => n.id === d.id || neighbors.has(n.id));

  // 链接同理
  linkSel.classed("dim", ...);
  linkSel.classed("highlight", ...);
});
```

#### 拖拽行为
```typescript
const dragBehavior = d3drag.drag<SVGGElement, GraphNode>()
  .on("start", (event, d) => {
    sim.alphaTarget(0.15).restart();  // 激活模拟
    d.fx = d.x;
    d.fy = d.y;
  })
  .on("drag", (event, d) => {
    d.fx = event.x;
    d.fy = event.y;
  })
  .on("end", (event, d) => {
    sim.alphaTarget(0.015);  // 回到环境运动
    d.fx = null;
    d.fy = null;
  });
```

#### 缩放/平移
```typescript
const zoomBehavior = d3zoom.zoom<SVGSVGElement, unknown>()
  .scaleExtent([0.2, 4])  // 缩放范围
  .on("zoom", (event) => {
    root.attr("transform", event.transform.toString());
  });
```

```typescript
sim.on("tick", () => {
```
